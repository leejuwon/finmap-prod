# 3. 핵심 개념 정리 섹션
## 3-1. 개념 정의
- {핵심 용어1}의 의미를 쉬운 말로 설명  
- 너무 교과서식보다, 실제 투자 상황 예시를 함께 제시  

## 3-2. 공식/계산 방식  
공식은 LaTeX 대신 **텍스트로** 풀어서 작성:

- 예:  
  - CAGR = (최종금액 / 초기금액)^(1 / 투자기간) - 1  
  - 샤프비율 = (포트폴리오 수익률 - 무위험 이자율) / 변동성  

가능하면 바로 아래에 **숫자 예시 1개** 붙여서 설명.
